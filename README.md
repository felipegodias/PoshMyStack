## Requirements
* PowerShell 7.x
```powershell
winget install Microsoft.PowerShell
```

## Install
```powershell
git clone git@github.com:felipegodias/PoshMyStack.git
cd PoshMyStack
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted
./Install.ps1
```
